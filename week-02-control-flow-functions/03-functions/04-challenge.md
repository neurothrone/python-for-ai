# 04. Challenge: Functions

## Challenge Tasks

1. Create a function `grade(score)` that returns `A`, `B`, `C`, or `F`.
2. Create a function `count_vowels(text)` that returns how many vowels are in a word.
3. Create a function `safe_divide(a, b)` that returns a message if `b` is `0`.

## Hint

Write and test one function at a time. Use simple test calls after each function.
