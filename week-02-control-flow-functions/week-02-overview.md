# Week 2 Overview: Control Flow and Functions

This page is the starting point for Week 2.

## Week Goal

By the end of Week 2, students should be able to make decisions with `if` statements, repeat actions with loops,
and write small functions with parameters and return values.

## Week 2 Completion Checklist

- I can use `if`, `elif`, and `else` to control program flow.
- I can use `for` and `while` loops for repeated tasks.
- I can avoid common loop mistakes like wrong indentation and infinite loops.
- I can define and call functions with parameters.
- I can return values from a function and use the result correctly.
