# 04. Challenge: Loops

## Challenge Tasks

1. Print the multiplication table for a number chosen by the user.
2. Ask for five numbers and print the largest number entered.
3. Use a loop to count how many even numbers are in a list.

## Hint

Start with a simple version first, then improve it step by step.
