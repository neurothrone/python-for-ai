# 02. Worked Examples: Loops

## Example A: Sum Numbers from a List

```python
numbers = [3, 5, 7]
total = 0

for n in numbers:
    total = total + n

print(total)
```

## Example B: Count Down with `while`

```python
count = 5

while count > 0:
    print(count)
    count = count - 1

print("Done")
```

## Example C: Input Until Correct Value

```python
answer = ""

while answer != "python":
    answer = input("Type python: ")

print("Correct")
```
