# 02. Course Roadmap (8 Weeks)

## Roadmap

| Week | Theme                                  |
|------|----------------------------------------|
| 1    | Introduction to the course and Python  |
| 2    | Control flow and functions             |
| 3    | OOP and Jupyter notebooks              |
| 4    | Git and data handling                  |
| 5    | Data visualization and ML introduction |
| 6    | Advanced machine learning              |
| 7    | Deep learning and neural networks      |
| 8    | Project work and AI ethics             |

## What Success Looks Like

By the end of the course, students can:

- Write clear Python code.
- Process and visualize data.
- Train and evaluate basic ML models.
- Use development tools and Git workflows.
- Discuss practical and ethical AI use.
