# 01. Theory: Input and Output

## Why This Matters

Input and output are the foundation of almost every program. If you can receive data, process it, and present clear
results, you can build useful tools in any domain, including AI and data analysis.

## Output with `print`

Use `print()` to show text or values.

```python
print("Welcome to Python")
```

## Input with `input`

Use `input()` to read user input.

```python
name = input("What is your name? ")
print("Hello, " + name)
```

## Important Type note

`input()` always returns text (`str`).

If you need numbers, convert input with `int()` or `float()`.
