# 04. Challenge: Input and Output

## Challenge Tasks

1. Build a short profile collector (name, age, country, favorite topic) and print a 2-line summary.
2. Ask for temperature in Celsius and print a rough Fahrenheit conversion.
3. Ask for three numbers and print their average.

## Hint

Use f-strings to keep output readable.

Quick Reference:

- [W3Schools: Python String Formatting and f-strings](https://www.w3schools.com/python/python_string_formatting.asp).
