# 02. Worked Examples: Data Types and Variables

## Example A: Student Profile

```python
name = "Sara"
age = 24
is_student = True
print(f"{name} is {age} years old. Student: {is_student}")
```

## Example B: Price Calculator

```python
price = 99.0
discount = 0.15
final_price = price * (1 - discount)
print(f"Final price: {final_price}")
```

## Example C: Type Check

```python
value = input("Enter a number: ")
number = int(value)
print(type(value))
print(type(number))
```
