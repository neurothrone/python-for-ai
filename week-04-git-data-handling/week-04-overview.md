# Week 4 Overview: Git and Data Handling

This page is the starting point for Week 4.

## Week Goal

By the end of Week 4, students should be able to use basic Git commands (`init`, `commit`, `push`, `pull`), read and
write small CSV and JSON files in Python, and use Pandas for simple data inspection, cleaning, transformation, and
saving cleaned results to a new file.

## Week 4 Completion Checklist

- I can create a local Git repository and make commits with clear messages.
- I can explain when to use `git push` and `git pull` in simple terms.
- I can read and write a small CSV file in Python.
- I can read and write a small JSON file in Python.
- I can load a CSV in Pandas, inspect columns, and handle simple null values.
- I can do one simple data transformation with Pandas.
- I can save cleaned data to a new file without overwriting the raw file.
