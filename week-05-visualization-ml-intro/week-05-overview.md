# Week 5 Overview: Visualization and ML Introduction

This page is the starting point for Week 5.

## Week Goal

By the end of Week 5, students should be able to create simple charts with Matplotlib and Seaborn, explain the basic
idea of features, target, train/test split, and build a first linear regression model with Scikit-learn.

## Week 5 Completion Checklist

- I can load a small dataset and inspect columns before plotting.
- I can choose a simple chart and add a clear title and axis labels.
- I can explain the difference between Matplotlib and Seaborn in simple terms.
- I can explain what a model is, what features are, and what a target is.
- I can split data into training and test parts in simple terms.
- I can fit a linear regression model, make predictions, and compare predicted values with actual values.
- I can keep raw data separate from outputs such as plots and prediction tables.
