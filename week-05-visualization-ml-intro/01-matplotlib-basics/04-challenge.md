# 04. Challenge: Matplotlib Basics

## Challenge Tasks

1. Create `bus_ridership.csv` with this content:
   ```text
   route,passengers
   Route A,120
   Route B,90
   Route C,150
   Route D,110
   ```
2. Load the file with Pandas.
3. Print `head()`, `columns`, and `dtypes`.
4. Create a bar chart with `route` on the x-axis and `passengers` on the y-axis.
5. Add the title `Bus Passengers by Route`.
6. Add clear axis labels.
7. Show the plot.
8. Write one simple interpretation of the chart in a `print()` statement.

## Expected Output Example

Possible printed sentence:

```text
Route C has the highest passenger count.
```

## Hint

Inspect the columns before plotting. If your chart window appears but shows no useful data, check the order of your
plotting code and labels.
