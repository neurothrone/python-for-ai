# 04. Challenge: Jupyter Basics

## Challenge Tasks

1. Create a Markdown header: `Notebook Check`.
2. Create a list `topics = ["Lists", "Classes"]`.
3. Use `append` to add `"Methods"`.
4. Use `insert` to add `"Jupyter"` at index `1`.
5. Use `remove` to remove `"Lists"`.
6. Print all values with a `for` loop.
7. Restart the kernel and run all cells.
8. Write one Markdown line about why run order matters.

## Expected Output Example

```text
Jupyter
Classes
Methods
```

## Hint

If values seem wrong, restart the kernel and run cells top to bottom.
