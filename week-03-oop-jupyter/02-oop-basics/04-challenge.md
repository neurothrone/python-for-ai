# 04. Challenge: OOP Basics

## Challenge Tasks

1. Create a class `Book` with `title` and `pages`.
2. Add method `is_long()` that returns `True` if pages are over 300.
3. Add method `summary()` that prints title and pages.
4. Create two books and test both methods.

## Expected Output Example

```text
False
True
Title: Short Book, Pages: 120
Title: Big Book, Pages: 450
```

## Hint

Use `return` in `is_long()` and `print()` in `summary()`.
