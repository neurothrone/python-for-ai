# Week 3 Overview: OOP and Jupyter

This page is the starting point for Week 3.

## Week Goal

By the end of Week 3, students should be able to work in Jupyter notebooks, create simple classes with attributes and
methods, and connect functions and state in one small class-based program.

## Week 3 Completion Checklist

- I can run notebook cells in the correct order and explain why order matters.
- I can create classes and objects with `__init__`, attributes, and methods.
- I can explain the difference between behavior (methods) and state (attributes and history lists).
- I can build a small class that stores calculation history.
- I can debug common mistakes: missing `self`, wrong method call, wrong indentation, and notebook state confusion.
