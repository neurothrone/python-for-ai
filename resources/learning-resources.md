# Learning Resources

Use this page as a central reference library for the course.

## Week 1 to 4 Core

- [W3Schools Python](https://www.w3schools.com/python/): Fast Python syntax lookups and beginner-friendly examples.
- [W3Schools Git](https://www.w3schools.com/git/): Quick Git command reference for early workflow practice.
- [LearnPython.org](https://learnpython.org/): Interactive practice to reinforce core concepts.

## Quick Reference

- [Python Documentation](https://docs.python.org/3/): Official and complete reference for Python behavior and standard
  library.

## Deeper Reading

- [Think Python](https://allendowney.github.io/ThinkPython/): Structured, book-style explanations for stronger
  fundamentals.
- [Real Python](https://realpython.com/): In-depth tutorials when you are ready to go beyond basics.
