# Learning Resources

Use this page as a central reference library for the course.

## Week 1 to 4 Core

- [W3Schools Python](https://www.w3schools.com/python/): Fast Python syntax lookups and beginner-friendly examples.
- [W3Schools Git](https://www.w3schools.com/git/): Quick Git command reference for early workflow practice.
- [LearnPython.org](https://learnpython.org/): Interactive practice to reinforce core concepts.

## Quick Cheat Sheets

### Python

- [Python Operators Reference (Course)](./python-operators-reference.md): Arithmetic and comparison operators with
  simple examples.
- [DataCamp Python Basics Cheat Sheet](https://media.datacamp.com/cms/python-basics-cheat-sheet-v4.pdf): Concise
  syntax and examples.
- [QuickRef Python](https://quickref.me/python): Fast one-page command and syntax recall.

### Git

- [GitHub Education Git Cheat Sheet](https://education.github.com/git-cheat-sheet-education.pdf): Common Git commands
  with practical workflow focus.
- [QuickRef Git](https://quickref.me/git): Compact Git command reference.

## Documentation (Official)

- [Python Documentation](https://docs.python.org/3/): The official Python guide; use it when you want trusted details
  or examples.

## Deeper Tutorials and Reading

- [Think Python](https://allendowney.github.io/ThinkPython/): Structured, book-style explanations for stronger
  fundamentals.
- [Real Python](https://realpython.com/): In-depth tutorials when you are ready to go beyond basics.
